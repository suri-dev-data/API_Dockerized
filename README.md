<h1>Dockerized FastAPI & MongoDB API</h1>
A clean and professional boilerplate for a high-performance, asynchronous REST API built with FastAPI and MongoDB, fully containerized using Docker.

<h1>Tech Stack</h1>
FastAPI (0.104.1): Modern Python web framework.

Motor (3.3.2): Asynchronous MongoDB driver for Python.

Pydantic (2.5.0): Data validation and settings management (V2).

Uvicorn (0.24.0): ASGI server for production and development.

Python-Multipart (0.0.6): Support for form data and file uploads.

Docker & Docker Compose: Complete environment orchestration.

<h1>Features</h1>
Asynchronous I/O: Fully non-blocking database operations.

Pydantic V2 engine: High-speed data serialization and validation.

Automated Documentation: Interactive Swagger UI and ReDoc.

Containerized Environment: One-command setup for both API and Database.
